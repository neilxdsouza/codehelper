Welcome! ProgHelp (Programmer Helper) is a tool written to help you build Web Applications requiring a Database Back-End very quickly. Think ERP-like applications.

It will build a web application for you as fast as you can write create table statements.

ProgHelp is still a work in progress, but you may find it useful

ProgHelp currently generates code in C++, targets the PostgreSQL database and uses the Wt toolkit as a web UI

    Here are some things that it does
  • Writes create table statements
  • Writes Stored Procedures
  • Creates Random data for the tables for testing
  • Writes a database extraction layer (DAL)
  • Writes a Business Logic Layer (BLL) which accesses the Database layer to retrieve data
  • Writes the Web User Interface (UI) forms which use the BLL to display, add, modify (CRU) data from the Web UI
  • If the input file has certain fields marked as search keys, it automatically implements the search facility right from the stored procedure to the UI (implementing at UI is still in progress, but will happen very soon)
  • Looks at the foreign key relationships and builds Dialog boxes to choose values
  • Writes the Makefile for you, so that you can test the web application quickly
